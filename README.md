# Yumma

Yumma is a fully designed multi-page web platform for listing and discovering PG accommodations, hotel rooms, restaurant services, and food offerings. It provides a complete responsive website with structured pages, listing displays, application and request forms, status pages, informational content, and supporting assets.

The site is built entirely with HTML, CSS, JavaScript, and static assets, making it easy to host on any static web hosting service.

---

## Overview

Yumma includes:

* A responsive homepage with hero section, highlights, and navigation
* Dedicated category pages for PG/hostels, hotels, restaurants, and food services
* Application and listing submission pages
* Data display pages
* Request, leave, and status workflow pages (front-end structured)
* Informational pages such as About, Help, Policy, Terms, and Feedback
* Login and user details pages
* Fully structured image assets and styling
* Static JavaScript-driven UI behaviors and form handling structure

This makes it suitable as a prototype, concept website, or static deployment for showcasing a service marketplace concept.

## Installation and Setup

1. Download or clone the repository.
2. Extract the project folder if downloaded as a ZIP.
3. Open `index.html` in any modern web browser.

No build step, server setup, or dependency installation is required.

---

## Usage

* Open `index.html` to access the main site.
* Use navigation links to explore:

  * PG listings
  * Hotels
  * Restaurants
  * Food service pages
* Use apply and display pages to view form and listing workflows.
* Review informational pages for platform details.
* Customize HTML, CSS, JavaScript, and images as needed.

---

## Notes

* This is a static website. Any submission or backend functionality would require custom integration.
* All styles are handled using `style.css`.
* Interactive behavior is implemented in `scripts.js`.
* Image libraries are organized inside the `images/` directory.
